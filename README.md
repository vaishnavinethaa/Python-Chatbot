# Python-Chatbot
# 🤖 Python Chatbot

A simple Question Answering chatbot built using Streamlit and Hugging Face Transformers.

## Features

- Ask questions about Python
- Uses DistilBERT Question Answering model
- Simple Streamlit interface
- Fast and lightweight

## Technologies Used

- Python
- Streamlit
- Hugging Face Transformers

## Installation

```bash
pip install streamlit transformers torch
```

## Run

```bash
streamlit run app.py
```

## Model

- distilbert-base-cased-distilled-squad
